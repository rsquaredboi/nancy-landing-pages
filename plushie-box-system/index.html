03 — The Solution
The System

One box.
Sticker does the work.

Same matte dusty pink box for every character. The collection sticker is the only thing that changes — applied during packing.

  • Box: Universal pink with holographic "?" — always the same
  • Badge: "SQUEEZE ME!" hot pink circle — Nancy brand marker
  • Sticker: Collection name + tagline + "1 of X" + silhouette icons
  • Reveal: Character pops out the top on open

08 — The Sticker System
Customisation Layer

10 stickers.
Infinite flexibility.

Each collection gets its own sticker. Printed in bulk sheets, applied during packing. New drop = new sticker, not new box.

  • Collection name + unique tagline front & center
  • "1 of X" creates set-completion FOMO
  • Tiny silhouette icons hint at characters
  • Foil / holographic variant = instant premium tier
  • Celeb collabs = just another sticker design
  • Character name on SIDE for known purchases
  • Taglines make each box feel unique & shareable
09 — Why This Scales
Unit Economics

Production simplified.

💰

Lower Unit Cost

One box mold = bulk production. Not 10 different designs with separate minimums.

⚡

Instant New Drops

New collection = print sticker sheet. Ship same week. Zero box lead time.

🏷️

Limited Editions

Foil sticker variant = premium tier. Same box, different finish. Instant hype.

📦

Warehouse Simple

One box SKU in stock. Stickers applied during pack. No 120 box variants.

🌍

Celeb Collabs

NK, future celebs — just another sticker. "CELEB COLLECTION", same pink box.

🎲

Blind Box Ready

The "?" is built in. Mystery energy works for both blind box and known purchases.
